Descrição Geral
Somelier Musical é um sistema web desenvolvido para criação, gerenciamento e votação de músicas em eventos musicais. O projeto permite que cantores organizem eventos, cadastrem suas playlists e disponibilizem uma página pública para votação.

FUNCIONALIDADES

Área do Cantor
• Criação de eventos com título, data, descrição e categoria
• Gerenciamento de playlists
• Publicação de eventos para permitir a votação
• Visualização dos resultados de cada evento

Área de Votação Pública
• Sistema de votação única por usuário
• Exibição de músicas de forma aleatória
• Confirmação e registro do voto

TECNOLOGIAS UTILIZADAS
• HTML5
• CSS3 com uso de variáveis
• JavaScript
